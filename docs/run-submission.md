# How to submit runs

!!! info "What is a run?"

    A run is the output of your instruction method generated by `geniie-lab`. A run is normally based on a particular method you developed. If you developed two instruction methods, you will have two `run-id`.

!!! note "Session Name"

    See [Common settings](https://github.com/geniie-lab/geniie-lab/blob/dev/docs/experiments/common_settings.md) to learn how to set a session name in your experiments.

    **Format**

    `[group-id]-[language]-[ir-model]-[run-id]`

    - `language`: `en` or `ja`
    - `ir-model`: `bm25` or `splade`
    - `run-id`: `1`, `2`, `3` (Your instruction method ID, up to 3)

    **Examples**

    |Group ID|Language|IR Model|Run ID|Session Name|
    |:--|:--|:--|:--|:--|
    |tsukuba48|English|BM25|1|`tsukuba48-en-bm25-1`|
    |tsukuba48|English|BM25|2|`tsukuba48-en-bm25-2`|
    |tsukuba48|English|SPLADAE|1|`tsukuba48-en-splade-1`|
    |tsukuba48|Japanese|BM25|1|`tsukuba48-ja-bm25-1`|

!!! abstract "Run File Name"

    `geniie-lab` save log files in `logs` folder: `.jsonl` and `.log`. Please change the name of `.jsonl` files to the following format.

    **Format**

    `[group-id]-[language]-[ir-model]-[run-id].jsonl`

!!! sucess "Submission"

    Send your jsonl files to `agenticinstruction-org at googlegroups dot com` before the submission deadline.
